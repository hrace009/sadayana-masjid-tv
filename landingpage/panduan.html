Miqotul Khoir TV Masjid

Panduan ini ditujukan untuk DKM, pengurus masjid, dan operator yang bertanggung jawab mengelola aplikasi Miqotul Khoir TV Masjid. Di sini Anda akan menemukan cara membuka menu pengaturan, menjalankan setup awal, memahami setiap menu konfigurasi, serta solusi atas pertanyaan umum yang sering muncul.

Panduan ini berlaku untuk dua mode penggunaan: Android TV dengan remote dan HP / tablet layar sentuh. Setiap bagian menjelaskan langkah-langkah untuk masing-masing perangkat secara terpisah agar tidak membingungkan.

Cara Masuk ke Pengaturan

Cara membuka menu pengaturan berbeda antara perangkat Android TV dengan remote dan HP atau tablet layar sentuh. Ikuti langkah yang sesuai dengan perangkat Anda.

Android TV / Remote

Dari layar utama, tekan salah satu tombol berikut pada remote TV untuk langsung membuka menu pengaturan:

  • OK / Select — tombol tengah D-Pad
  • Enter — keyboard eksternal
  • Back / Escape — tombol kembali remote
  • Play/Pause — tombol media remote

Semua tombol di atas langsung membuka pengaturan — tidak perlu mengetuk layar terlebih dahulu seperti di perangkat sentuh.

HP / Tablet Touchscreen

Di perangkat layar sentuh, membuka pengaturan membutuhkan dua langkah karena aplikasi menyembunyikan ikon settings secara default:

  1. Ketuk di mana saja pada layar utama untuk memunculkan ikon ⚙
  2. Ikon ⚙ akan muncul selama 5 detik, lalu hilang otomatis
  3. Ketuk ikon ⚙ tersebut sebelum 5 detik habis untuk masuk ke pengaturan

Tips: Jika ikon ⚙ tidak muncul, pastikan Anda mengetuk area layar utama — bukan tombol lain. Ketuk satu kali saja, lalu segera ketuk ikon ⚙ yang muncul sebelum 5 detik habis.

Setelah Pengaturan Terbuka

Baik melalui remote TV maupun layar sentuh, alur selanjutnya adalah sama:

  • Jika PIN tidak aktif: menu pengaturan langsung terbuka tanpa layar PIN.
  • Jika PIN aktif: halaman input PIN 6 digit muncul terlebih dahulu. Masukkan PIN yang benar untuk melanjutkan, atau tekan Kembali untuk membatalkan. Lihat Panduan PIN.

Cara Navigasi Menu Pengaturan

Menu pengaturan menggunakan layout dua panel: daftar kategori di panel kiri dan isi pengaturan di panel kanan. Navigasi berbeda antara remote TV dan layar sentuh.

Android TV / Remote
  • Gunakan ↑ ↓ (Atas/Bawah) untuk berpindah antar kategori di panel kiri
  • Tekan OK / Select untuk memilih kategori — isi pengaturan tampil seketika di panel kanan
  • Tekan → (Kanan) untuk berpindah fokus ke panel kanan dan berinteraksi dengan kontrol pengaturan
  • Navigasi ke tombol "Tutup Pengaturan" di bagian bawah panel kiri untuk menutup dan kembali ke layar utama

Saat pertama terbuka, Identitas Masjid sudah terpilih dan fokus berada di kategori pertama.

HP / Tablet
  • Ketuk nama kategori di panel kiri untuk membuka isinya
  • Isi pengaturan langsung tampil di panel kanan
  • Tombol "Tutup Pengaturan" ada di bagian bawah panel kiri

  • Di Step 3, dropdown kota hanya akan muncul setelah provinsi dipilih terlebih dahulu.

  • Jika ingin mengulang setup, gunakan menu Reset Data di pengaturan.
1

Selamat Datang

Layar pertama menampilkan logo aplikasi, judul Miqotul Khoir TV, dan keterangan singkat bahwa wizard akan memandu konfigurasi awal masjid.

Aksi: Tekan OK / Select (remote TV) atau ketuk tombol "Mulai Setup" (layar sentuh). Tombol ini sudah difokuskan otomatis — cukup tekan Enter di keyboard eksternal.

2

Identitas Masjid

Isi dua kolom teks menggunakan keyboard yang muncul:

  • Nama Masjid — nama resmi masjid, contoh: Masjid Raya Bandung.
  • Alamat Lengkap — alamat jalan dan kelurahan, contoh: Jl. Asia Afrika No. 123.

Aksi: Tekan "Selanjutnya" untuk lanjut. Kedua kolom wajib diisi — pesan validasi akan muncul di bawah form jika ada yang kosong. Tekan "Kembali" untuk kembali ke langkah 1.

3

Pilih Lokasi

Dua dropdown harus dipilih secara berurutan:

  • Provinsi — pilih provinsi tempat masjid berada.
  • Kota / Kabupaten — dropdown ini baru muncul setelah provinsi dipilih. Pilih kota atau kabupaten yang paling dekat dengan lokasi masjid.

Aksi: Setelah kota dipilih, tekan "Selanjutnya" untuk menuju pratinjau. Tekan "Kembali" untuk kembali ke langkah 2.

4

Konfirmasi & Simpan

Layar pratinjau menampilkan ringkasan semua data beserta jadwal sholat hari ini berdasarkan kota yang dipilih. Periksa kembali:

  • Nama masjid dan alamat sudah sesuai.
  • Kota / kabupaten sesuai lokasi masjid.
  • Jadwal sholat yang tampil masuk akal untuk lokasi tersebut.

Aksi: Tekan "Simpan & Selesai" untuk menyimpan semua data ke perangkat dan menutup wizard. Tekan "Kembali" untuk memperbaiki data sebelum menyimpan.

Jika PIN Aktif

Fitur PIN melindungi menu pengaturan agar hanya operator yang berwenang yang dapat mengubah konfigurasi masjid.

Kondisi Saat Membuka Settings

  • PIN tidak aktif: settings langsung terbuka tanpa halaman PIN
  • PIN aktif: halaman PIN muncul — masukkan 6 digit PIN yang benar
  • PIN salah: muncul pesan error selama sebentar, lalu bisa coba lagi
  • Tombol Kembali membatalkan dan kembali ke layar utama

Mengelola PIN

Buat, ubah, atau nonaktifkan PIN melalui menu pengaturan:

Menu Pengaturan → Keamanan (PIN)

Jika PIN terlupa, lihat FAQ untuk langkah pemulihan.

Identitas Masjid

Daftar menu

Nama dan alamat masjid yang ditampilkan di pojok kiri atas layar utama sebagai identitas visual di setiap mode tampilan.

Pengaturan Penting
  • Nama Masjid — teks bebas, wajib diisi (minimal 3 karakter)
  • Alamat Lengkap — teks bebas, boleh dikosongkan
  • Tekan tombol "Simpan" untuk mengonfirmasi perubahan
Dampak di Layar Utama

Nama dan alamat tampil di header pojok kiri atas di semua state layar (Standby, Pre-Adzan, Adzan, Iqomah).

Catatan Android TV: Tekan OK / Enter pada tombol edit via D-Pad untuk memunculkan keyboard virtual, lalu ketik nama atau alamat.

Koreksi Waktu (Ihtiyat)

Daftar menu

Penyesuaian offset ±30 menit per waktu sholat untuk kebutuhan ihtiyat (kehati-hatian) atau koreksi jadwal lokal masjid.

Pengaturan Penting
  • Offset Subuh — rentang −30 s.d. +30 menit
  • Offset Syuruq — rentang −30 s.d. +30 menit
  • Offset Dhuha — rentang −30 s.d. +30 menit
  • Offset Dzuhur — rentang −30 s.d. +30 menit
  • Offset Ashar — rentang −30 s.d. +30 menit
  • Offset Maghrib — rentang −30 s.d. +30 menit
  • Offset Isya — rentang −30 s.d. +30 menit
Dampak di Layar Utama

Perubahan offset langsung memengaruhi jadwal sholat yang tampil di layar utama dan seluruh transisi state (Pre-Adzan, Adzan, Iqomah).

Catatan: Aplikasi sudah menggunakan standar Kemenag RI (SIHAT) dengan ihtiyat bawaan +2 menit. Ubah offset ini hanya jika ada keputusan pengurus atau kesepakatan jamaah untuk penyesuaian tambahan.

Durasi Iqomah

Daftar menu

Jeda waktu antara selesainya adzan dan dimulainya iqomah untuk setiap waktu sholat wajib.

Pengaturan Penting
  • Subuh — rentang 1–30 menit
  • Dzuhur — rentang 1–30 menit
  • Jum'at — rentang 1–30 menit
  • Ashar — rentang 1–30 menit
  • Maghrib — rentang 1–30 menit
  • Isya — rentang 1–30 menit
Dampak di Layar Utama

Layar menampilkan countdown iqomah (hitung mundur) setelah adzan selesai. Durasi countdown ini mengikuti nilai yang diatur per waktu sholat.

Catatan: Waktu Jum'at umumnya diatur lebih panjang karena khutbah berlangsung sebelum iqomah dimulai.

Pengaturan Dhuha

Daftar menu

Menentukan kapan waktu Dhuha mulai ditampilkan, dihitung sejumlah menit setelah waktu Syuruq.

Pengaturan Penting
  • Offset Dhuha — rentang 10–30 menit setelah Syuruq
Dampak di Layar Utama

Waktu Dhuha di jadwal layar utama bergeser maju sejumlah offset menit setelah waktu Syuruq.

Durasi Tampilan

Daftar menu

Mengatur lama tiap state tampilan layar utama serta kalibrasi tanggal Hijriyah agar sesuai dengan kalender resmi masjid.

Pengaturan Penting
  • Pre-Adzan (Persiapan) — 5–30 menit sebelum adzan
  • Tampilan Waktu Adzan — 60–600 detik (kelipatan 10)
  • Durasi Layar Hitam (Saat Sholat) — 10–45 menit (kelipatan 5)
  • Durasi Layar Hitam Jum'at — 10–90 menit (kelipatan 5)
  • Kalibrasi Tanggal Hijriyah — −2 s.d. +2 hari
Dampak di Layar Utama

Transisi Standby → Pre-Adzan → Adzan → Iqomah → Sholat mengikuti durasi yang dikonfigurasi. Layar hitam saat sholat aktif sesuai waktu yang diatur.

Catatan: Gunakan "Kalibrasi Tanggal Hijriyah" jika tanggal Hijriyah yang tampil berbeda 1–2 hari dari kalender resmi masjid. Nilai positif memajukan tanggal, nilai negatif memundurkan.

Alarm Tanda Waktu

Daftar menu

Membunyikan suara tanda singkat beberapa detik sebelum adzan dimulai dan/atau sebelum iqomah selesai sebagai pengingat bagi jamaah.

Pengaturan Penting
  • Alarm Pre-Adzan — toggle aktif/nonaktif + countdown 5–15 detik sebelum adzan
  • Alarm Pre-Iqomah — toggle aktif/nonaktif + countdown 5–15 detik sebelum iqomah selesai
Dampak di Layar Utama

Bunyi audio muncul dari speaker perangkat Android TV sesuai waktu yang ditentukan, tanpa mengubah tampilan layar.

Catatan: Stepper countdown hanya aktif saat toggle terkait dinyalakan. Pastikan volume speaker perangkat tidak dalam kondisi mute.

Running Text

Daftar menu

Teks berjalan di footer layar utama — cocok untuk pengumuman, jadwal kegiatan, atau informasi harian masjid.

Pengaturan Penting
  • Area Input Teks — teks bebas, mendukung teks panjang multiline
  • Preview real-time tersedia di bagian atas section
  • Perubahan disimpan setelah menekan Done / Enter
Dampak di Layar Utama

Teks berjalan terus-menerus di bagian bawah layar utama pada semua mode tampilan (Standby, Pre-Adzan, Adzan, Iqomah).

Catatan Android TV: Tekan OK / Enter pada area teks via D-Pad untuk memunculkan keyboard virtual. Teks baru langsung terlihat di preview sebelum disimpan.

Keamanan (PIN)

Daftar menu

Proteksi menu pengaturan dengan PIN 6 digit agar hanya operator berwenang yang dapat mengubah konfigurasi aplikasi.

Pengaturan Penting
  • Tombol "Buat PIN" — jika PIN belum pernah dibuat
  • Tombol "Ubah PIN" — jika PIN sudah aktif
  • Tombol "Nonaktifkan PIN" — hanya muncul jika PIN sedang aktif
Dampak di Layar Utama

Saat PIN aktif, setiap kali membuka pengaturan user akan diminta memasukkan PIN 6 digit di halaman PIN gate sebelum masuk ke menu.

Catatan: Operator yang sudah masuk melewati PIN gate tidak perlu memasukkan PIN lagi selama sesi pengaturan berjalan. Lihat juga panduan Jika PIN Aktif.

Informasi Kas

Daftar menu

Tampilkan data keuangan masjid di layar utama sebagai bentuk transparansi kepada jamaah.

Pengaturan Penting
  • Toggle Aktifkan Informasi Kas
  • Saldo Kas — angka Rupiah, diisi manual
  • Pemasukan Periode Ini — angka Rupiah, diisi manual
  • Pengeluaran Periode Ini — angka Rupiah, diisi manual
Dampak di Layar Utama

Informasi kas masjid tampil di layar utama saat mode standby ketika fitur diaktifkan.

Catatan: Nilai diformat otomatis ke "Rp X.XXX.XXX". Data diisi manual oleh pengurus — tidak terhubung ke sistem keuangan manapun.

Kata Mutiara

Daftar menu

Tampilkan ayat Al-Quran atau hadits pilihan secara periodik di layar utama sebagai pengingat spiritual bagi jamaah.

Pengaturan Penting
  • Toggle Aktifkan Kata Mutiara
  • Tampil Setiap — 5–30 menit (kelipatan 5)
  • Lama Tampil — 1–10 menit per sesi
  • Jam Aktif — Dari Jam:Menit s.d. Sampai Jam:Menit
  • Mode Tampil — Urut atau Acak (radio)
  • Pilih Konten — checklist 11 item (5 ayat Al-Quran + 6 hadits)
  • Tombol Pratinjau konten terpilih
Dampak di Layar Utama

Layar full-screen Kata Mutiara muncul periodik, menggantikan layar standby selama durasi yang diatur, sesuai jam aktif.

Catatan: Kata Mutiara tidak tampil di luar jam aktif yang ditentukan. Jika tidak ada item yang dipilih di daftar checklist, fitur tidak akan menampilkan konten meskipun toggle dalam kondisi ON.

Slideshow Pengumuman

Daftar menu

Tampilkan gambar pengumuman kegiatan masjid secara periodik di layar utama. Mendukung hingga 3 slot gambar yang dapat dikelola secara independen.

Pengaturan Penting
  • Toggle Aktifkan Slideshow Pengumuman
  • Jadwal Aktif — Jam:Menit Mulai s.d. Jam:Menit Berakhir
  • Interval Antar Slot — jeda antar penampilan slideshow (menit)
  • Durasi Per Slot — lama satu sesi slideshow tampil (menit)
  • Durasi Per Gambar — lama tiap gambar tampil (detik)
  • 3 slot gambar: Pilih / Ganti / Hapus per slot
  • Tombol Pratinjau tersedia per slot gambar
Dampak di Layar Utama

Gambar pengumuman tampil full-screen secara periodik sesuai jadwal aktif, menggantikan layar standby selama durasi yang diatur.

Catatan: Toggle otomatis nonaktif jika semua slot gambar dihapus. Gambar disimpan lokal di perangkat — tidak memerlukan koneksi internet.

Jadwal Imam

Daftar menu

Tampilkan jadwal imam sholat berjamaah harian secara periodik di layar utama agar jamaah mengetahui siapa imam pada setiap waktu sholat.

Pengaturan Penting
  • Toggle Aktifkan Jadwal Imam
  • Toggle Kunci Jadwal — cegah perubahan jadwal yang tidak disengaja
  • Durasi Tampil — lama jadwal imam tampil di layar utama
  • Grid jadwal 7 hari (Senin–Minggu) × masing-masing waktu sholat
  • Tambah / Edit / Hapus data imam per hari dan waktu
Dampak di Layar Utama

Layar jadwal imam tampil periodik selama durasi yang diatur. Hari yang ditampilkan menyesuaikan otomatis dengan hari aktual.

Catatan: Aktifkan kunci jadwal setelah semua data imam selesai diisi untuk mencegah perubahan tidak disengaja via D-Pad. Nonaktifkan kunci terlebih dahulu sebelum memperbarui jadwal.

Mode Hemat Daya

Daftar menu

Alihkan layar ke tampilan minimal (jam digital + jadwal Subuh) pada jam-jam malam untuk menghemat daya dan mencegah burn-in layar.

Pengaturan Penting
  • Toggle Aktifkan Mode Hemat Daya
  • Jam Mulai + Menit Mulai — waktu layar beralih ke mode minimal
  • Jam Berakhir + Menit Berakhir — waktu kembali ke standby normal
Dampak di Layar Utama

Layar beralih ke mode minimal gelap dengan jam digital dan jadwal Subuh. Otomatis kembali ke standby normal setelah jam berakhir.

Catatan: Mode ini mendukung rentang lintas tengah malam — contoh: 22:00 s.d. 04:30. Konfigurasi dinonaktifkan (abu-abu) saat toggle dalam kondisi OFF.

Reset Data

Daftar menu

Hapus semua pengaturan dan kembalikan aplikasi ke kondisi bawaan pabrik. Gunakan jika ingin mengkonfigurasi ulang dari awal atau pindah ke masjid lain.

Pengaturan Penting
  • Tombol Reset — memunculkan dialog konfirmasi sebelum eksekusi
  • Data yang dihapus: kota tersimpan, koreksi waktu, dan identitas masjid
  • Tekan "Batal" di dialog untuk membatalkan tanpa efek apapun
Dampak di Layar Utama

Setelah reset berhasil, aplikasi menavigasi ke Setup Wizard untuk mengisi ulang semua konfigurasi dari langkah pertama.

⚠️ Peringatan: Tindakan ini tidak dapat dibatalkan setelah dikonfirmasi. Catat konfigurasi penting (nama masjid, kota, koreksi waktu) sebelum melakukan reset.

Pertanyaan Umum & Kendala

Kumpulan solusi untuk masalah yang paling sering dialami operator masjid saat menggunakan Miqotul Khoir TV Masjid.

Tekan tombol OK / Select pada remote TV dari layar utama untuk langsung membuka menu pengaturan. Anda juga dapat menggunakan tombol Enter, Escape (Back), atau Play/Pause. Lihat panduan cara masuk ke pengaturan untuk detail lebih lanjut.

Ketuk di mana saja pada layar utama untuk memunculkan ikon ⚙. Ikon akan tampil selama 5 detik, lalu hilang otomatis. Segera ketuk ikon ⚙ tersebut sebelum 5 detik berakhir. Jika berhasil masuk, akan langsung diarahkan ke halaman PIN (jika PIN aktif) atau ke menu pengaturan.

Jika PIN tidak bisa diingat, Anda dapat menghapus data aplikasi melalui menu Pengaturan Android → Aplikasi → Miqotul Khoir TV Masjid → Hapus Data. Setelah data dihapus, wizard setup akan muncul kembali dan Anda dapat memulai dari awal tanpa PIN. Lihat panduan menu Reset Data untuk langkah-langkah lebih detail.

Buka menu pengaturan, pilih Reset Data, dan konfirmasi reset. Setelah reset berhasil, aplikasi otomatis kembali ke layar awal dan menampilkan wizard setup agar Anda dapat mengulang pengisian identitas masjid dan lokasi dari awal.

Pastikan kota yang dipilih di wizard setup atau pengaturan sudah sesuai dengan lokasi masjid. Jika masih ada selisih, Anda dapat menyesuaikan offset menit per waktu sholat melalui menu Koreksi Waktu (Ihtiyat). Waktu sholat dihitung menggunakan metode Kemenag SIHAT (Subuh 20°, Isya 18°, Ihtiyat +2 menit).
